# 报销管理制度

## 适用范围
本制度适用于公司所有员工的差旅、办公、招待等费用的报销。

## 可报销费用类型
| 类型 | 说明 | 需提供凭证 |
|------|------|----------|
| 差旅费 | 机票、火车票、住宿费 | 发票 + 行程单 |
| 办公费 | 办公用品、软件订阅 | 发票 + 购物清单 |
| 招待费 | 客户招待、团队聚餐 | 发票 + 招待说明 |
| 交通费 | 打车、地铁、公交 | 发票或行程记录 |

## 报销流程
1. 登录财务系统，填写「费用报销单」
2. 上传发票照片或 PDF（清晰、完整）
3. 填写费用明细：
   - 费用类型
   - 发生日期
   - 金额
   - 事由说明
4. 提交审批：直属主管 → 财务审核 → 出纳付款
5. 预计到账时间：审批通过后 3-5 个工作日

## 报销标准
### 差旅住宿标准
| 城市级别 | 标准（元/晚） |
|---------|----------------|
| 一线城市（北上广深） | 500 |
| 新一线城市 | 400 |
| 其他城市 | 300 |

### 招待费标准
- 普通招待：人均不超过 150 元
- 重要客户：人均不超过 300 元（需副总审批）

## 不可报销的费用
- 个人消费（如零食、私人物品）
- 超标准部分（按标准报销，超出自理）
- 无发票的费用（特殊情况需书面说明）

## 常见问题
**Q：发票丢失了怎么办？**
A：发票丢失无法报销，请务必妥善保管发票原件。

**Q：报销单被退回怎么办？**
A：查看退回原因，修改后重新提交。
