# IT 设备申请指南

## 适用范围
本指南适用于公司员工申请办公电脑、显示器、键盘鼠标等 IT 设备。

## 可申请设备清单
| 设备类型 | 申请条件 | 审批流程 |
|---------|---------|----------|
| 办公电脑（笔记本/台式机） | 入职时统一发放，损坏可更换 | 直属主管 → IT 部门 |
| 显示器（第二屏） | 入职满 3 个月可申请 | 直属主管审批 |
| 键盘、鼠标、耳机 | 随时可申请 | 直接提交 IT 部门 |
| 移动硬盘、U 盘 | 工作需求 | 直属主管审批 |
| 办公软件授权 | 工作需求 | 直属主管 → IT 部门 |

## 申请流程
1. 登录 IT 服务台（http://it.company.com）
2. 点击「设备申请」，选择设备类型
3. 填写申请理由（请详细说明工作用途）
4. 提交审批
5. 审批通过后，IT 部门会在 3 个工作日内联系您领取设备

## 设备更换
### 电脑更换
- 使用年限满 3 年可申请更换
- 设备故障可申请提前更换（需 IT 部门检测确认）
- 更换时需交回旧设备，IT 部门进行数据清除

### 配件更换
- 键盘、鼠标等配件损坏可随时申请更换
- 需将损坏配件交回 IT 部门

## 设备使用规范
- 公司设备仅用于工作用途，不得转借他人
- 离职时需交回所有公司设备
- 设备损坏需说明原因，人为损坏需赔偿

## 常见问题
**Q：新员工入职时发什么电脑？**
A：默认发放公司标准办公笔记本（ThinkPad T 系列），特殊需求可在入职前沟通。

**Q：可以自己买设备然后报销吗？**
A：不可以，所有 IT 设备需通过 IT 部门统一采购和发放。

**Q：Mac 电脑可以申请吗？**
A：可以，设计部和开发部员工默认发放 Mac，其他部门需特殊申请。
